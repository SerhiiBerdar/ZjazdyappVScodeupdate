# Analýza Vytaženosti Zjazdov

Dashboard na analýzu pohybu KLT prepraviek cez stanice v čase.

## Spustenie lokálne

```bash
npm install
npm run dev
```

## Deploy na Vercel

1. Pushni na GitHub
2. Importuj repozitár na [vercel.com](https://vercel.com)
3. Vercel automaticky detekuje Vite — klikni **Deploy**

## Použitie

1. V Exceli označ všetky riadky (3 stĺpce: **Čiarový kód** · **Stanica** · **Dátum/čas**)
2. Skopíruj `Ctrl+C`
3. Vlož do poľa v záložke Dashboard `Ctrl+V`
4. Klikni **▶ Analyzovať*
